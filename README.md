# DS&A — Asymptotic Notation: Proofs and Counter-examples

## About the course

This repository is part of my coursework for **Data Structures and Algorithms**, a university course on algorithm analysis and design. Topics include asymptotic analysis, sorting, balanced search trees, hashing, graphs and shortest-path algorithms, dynamic programming, and amortized analysis — with both written theoretical exercises and a final programming project.

## About this assignment

This is the first written exercise of the course. Each question asks to prove or disprove a claim about asymptotic bounds (Big-O, Big-Omega, Big-Theta) on asymptotically-positive integer-domain functions, strictly from the epsilon/N-style definitions — no limits are allowed. The solutions walk through definitions, construct explicit witnesses, and in a few cases produce concrete counter-examples.
